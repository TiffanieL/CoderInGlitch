Original CODER website


Things to make. Stuff to learn.

Coder Projects are simple, fun, and sneakily educational things that can all be made with an online environment called Glitch. Browse the list below to see what catches your eye, or if you feel inspired, contribute your own project to the site.

Heads up!

These projects were originally created by Google Creative Labs - Coder for Pi which used a Raspberry Pi and a special Coder App for development. This particular website is a remix of that, to bring you the same tutorials (and hopefully new tutorials later on!) but without the need for a Raspberry Pi. That means the instructions in the tutorials needed to be converted to use an online environment called Glitch where you can write your code. If the "Open Project" button is purple that means it is converted and you're good to go! If you feel confident in being able to read the original tutorials and still use Glitch click on the yellow "Open Project" button and it will take you to the original tutorial.

More About Coder Projects

Coder Projects are Coder apps and instructional step-by-step guides designed to get new programmers excited about making things with code. They range from introductory lessons on the basic web stuff Coder utilizes (HTML, CSS, Javascript) to more complex projects that show what code can do.

Coder projects are perfect for anyone new to coding, or helping someone else learn. Each project is designed to be a jumping off point that will leave you with something fun and hopefully give you ideas for projects of your own.


Need some more help?
Coder projects are designed to be fun and educational excercises but they won’t replace full-blown lessons and web development courses. If you’d like to dig deeper there are tons of great resources out there for learning the languages of the web. Codecademy and Khan Academy are two of our favorites, and you can find lots more at Code.org.
